# Beauty LP

美容脱毛サービスをイメージしたランディングページ（LP）です。  
HTML / CSS / JavaScript を用いて制作しました。

---

#公開URL
https://atom2344.github.io/beauty.Lp/

# Features

- レスポンシブ対応
- FV（ファーストビュー）を意識したLPデザイン
- Flexbox を用いたレイアウト
- メディアクエリによる画面サイズ調整
- 背景画像・クーポン画像を使用したビジュアル設計
- ハンバーガーメニュー実装中

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
